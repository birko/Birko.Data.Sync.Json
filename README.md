# Birko.Data.Sync.Json

JSON file-based sync knowledge item using System.Text.Json for the Birko Framework.

## Installation

```bash
dotnet add package Birko.Data.Sync.Json
```

## Dependencies

- Birko.Data.Core (AbstractModel)
- Birko.Data.Stores (store interfaces, Settings)
- Birko.Data.Sync
- Birko.Data.JSON
